# Project Description for Resume

**Project Name:** NextPro Blog Platform

**Description:**
Designed and developed a full-stack, real-time blogging platform using **Next.js 16** and **React 19**. Implemented a high-performance backend with **Convex** for real-time data syncing (live presence indicators, instant comments) and **Prisma** with PostgreSQL for robust data management.

**Key Achievements:**
*   Built a responsive, "premium-feel" UI using **Tailwind CSS v4** and **Radix UI/Shadcn** components, featuring dark mode and smooth Framer Motion animations.
*   Integrated secure authentication using **Better Auth**.
*   Engineered a seamless content creation workflow with a rich text editor and image handling.
*   Optimized for performance and SEO, achieving fast load times and excellent core web vitals.

**Tech Stack:** Next.js 16, React 19, Convex DB, TypeScript, Tailwind CSS, Prisma, PostgreSQL.
